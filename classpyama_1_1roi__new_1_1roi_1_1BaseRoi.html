Detailed Description

Base class for representing a region-of-interest (ROI).

group is None or a group. Will be stored as weakref. color is None or a string of a hex RGB color (#rrggbb) visible is a boolean flag whether to display the ROI.

If the color or visible properties are None, their values are inherited from the group.
